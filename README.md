## About The Project

<div align="center">
  <img alt="Screenshot of program" src="radioinfo.png" />
</div>

A program that fetches data from SR's public API and show the information about radio programs - both for the previous adn next 24 hours. The program can 
play audio from the radio programs and uses threads.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [Java Swing](https://docs.oracle.com/javase/tutorial/uiswing/)
* [SR API](https://sverigesradio.se/oppetapi)
